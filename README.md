# prototype_project

A new Flutter project.

## Project Building
1. Follow the [Flutter Getting Started Doc](https://docs.flutter.dev/get-started/quick) to your preferred operating system.
2. Open you preferred terminal.
	1. Change directories to an appropriate location. 
	2. Enter `git clone https://github.com/FinleyConway/prototype_project.git`
	3. Enter `cd prototype_project && code .`
    4. Within the code terminal enter: `flutter run`
